# Books RAG Assistant

An AI question-answering assistant over a book catalogue, built with a
Retrieval-Augmented Generation (RAG) pipeline. Data is scraped from
[books.toscrape.com](https://books.toscrape.com) (a public sandbox site
built for scraping practice).

## What it does

Ask natural-language questions about the book catalogue (e.g. *"cheap
travel books rated 4 stars or higher"*) and get an answer grounded only
in the retrieved catalogue data, generated by the Google Gemini API.

## Pipeline

```
scrape -> chunk -> embed/index -> [query rewrite -> classify -> extract filters]
       -> hybrid retrieve (dense + BM25) -> Gemini answer
```

## Setup (Windows)

1. Install Python 3.10+ from python.org (check "Add Python to PATH").
2. Open this folder in VS Code, then open a terminal (`Terminal > New Terminal`).
3. Create and activate a virtual environment:
   ```
   python -m venv venv
   venv\Scripts\activate
   ```
4. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
5. Set up your Gemini key:
   - Copy `.env.example` to `.env`
   - Get a free key from https://aistudio.google.com/apikey
   - Paste it into `.env`

## Run (in order)

```
python src/scrape.py     # collect ~1000 books  -> data/books_raw.json
python src/chunk.py       # build chunks         -> data/chunks.json
python src/index.py       # embed + build index  -> data/faiss.index
streamlit run app.py      # launch the GUI
```

The first three steps only need to be run once. After that, just launch
the Streamlit app.
